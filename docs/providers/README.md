# LLM Providers

Ghost supports 20+ LLM providers. Pick one during setup or switch anytime.

## Quick Links

- **[Picking a Provider](./overview.md)** — Decision tree and provider comparison
- **[Custom Models](./CUSTOM_MODELS.md)** — Ollama, vLLM, LM Studio setup

## Common Tasks

```bash
bun run dev onboard                          # Change your provider / model
bun run dev providers                        # List all available providers
bun run dev providers --models <provider>    # List models for a provider
bun run dev doctor                           # Test your current provider
```
