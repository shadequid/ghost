# LLM Providers

Ghost supports 20+ LLM providers. Pick one during setup or switch anytime.

## Quick Links

- **[Picking a Provider](./overview.md)** — Decision tree and provider comparison
- **[Custom Models](./CUSTOM_MODELS.md)** — Ollama, vLLM, LM Studio setup

## Common Tasks

```bash
ghost onboard                          # Change your provider / model
ghost providers                        # List all available providers
ghost providers --models <provider>    # List models for a provider
ghost doctor                           # Test your current provider
```
